

## ComplianceRAG - Assistant Conformité FINMA

### Page unique : Interface de Chat

**Header**
- Fond bleu marine foncé (#0F172A), logo 🏦 + titre "ComplianceRAG" en blanc, sous-titre "Assistant Conformité FINMA" en gris clair

**Zone de chat (centre, scrollable)**
- Messages utilisateur : bulles bleues alignées à droite
- Messages agent : bulles gris foncé alignées à gauche avec icône 🤖, section "📄 Sources :" en italique sous le texte
- Indicateur de chargement animé ⏳ avec texte "Analyse en cours..." quand l'agent "réfléchit"

**Barre d'input (bas, fixe)**
- Champ texte avec placeholder "Posez votre question de conformité..."
- Bouton "Envoyer" bleu
- Envoi aussi possible avec la touche Entrée

**Fonctionnement**
- Messages de démonstration pré-remplis pour montrer le design
- Réponses simulées de l'agent avec un délai de 1-2 secondes et indicateur de chargement
- Quelques réponses fictives sur la conformité FINMA avec des sources exemple
- Design responsive, palette sombre professionnelle style bancaire
